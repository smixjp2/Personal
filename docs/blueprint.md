# **App Name**: Life Architect

## Core Features:

- Habit Tracking: Track daily, monthly, and yearly habits with a progress view.
- Goal Setting: Set and manage goals across different project types (courses, personal, professional).
- Progress Visualization: Visualize progress through charts and graphs to monitor performance.
- AI-Powered Task Prioritization Tool: Receive suggestions, powered by AI, on prioritizing daily tasks to help meet set goals and deadlines.
- Automated Task Generation: Use generative AI to create tasks necessary for project completion, to act as an accountability partner.
- Calendar Integration: Integrate the app with existing calendars for consolidated view of tasks and schedules.
- Reminders and Notifications: Set reminders and notifications to stay on track with habits and goals.

## Style Guidelines:

- Primary color: Soft blue (#A7D1AB) for tranquility and focus.
- Background color: Very light cyan (#E0FFFF) for a calm, uncluttered feel.
- Accent color: Light purple (#BCA0DC) to highlight important actions.
- Body and headline font: 'PT Sans' sans-serif font, to keep an inviting feel
- Code font: 'Source Code Pro' for displaying code snippets, to aid debugging, or document configuration
- Simple, outline-style icons to represent different habits and goal categories.
- Clean and organized layout with clear visual hierarchy for easy navigation.
- Subtle animations to provide feedback on interactions and show progress.
